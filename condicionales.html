Condicionales en Java

Aprende a usar estructuras condicionales if y switch para tomar decisiones.

Estructura If-Else

Ejemplo de uso de if para validar la edad de una persona:

if (edad >= 18) {  
    System.out.println("Eres mayor de edad");  
} else {  
    System.out.println("Eres menor de edad");  
}
