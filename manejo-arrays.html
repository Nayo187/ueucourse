Manejo ArrayList en Java

Aprende a trabajar con arrays y ArrayList para almacenar múltiples valores.

Objetivo de la Clase: (ArrayList con Interfaz Gráfica - JTable en Java Swing)

El estudiante aplicará el uso de la clase ArrayList para almacenar, agregar y eliminar datos dinámicamente en una interfaz gráfica creada con Java Swing, mostrando los resultados en una tabla (JTable).

ArrayList en Java

Un ArrayList es una clase en Java que permite almacenar una lista dinámica de objetos. A diferencia de los arrays, un ArrayList puede cambiar de tamaño automáticamente.

Sintaxis Básica

ArrayList<TipoDeDato> nombreLista = new ArrayList<>();

Ejemplo:

ArrayList<String> nombres = new ArrayList<>();

Propiedades Básicas

  • Tamaño dinámico: No es necesario definir un tamaño fijo.
  • Almacena objetos: Solo puede almacenar objetos, no tipos primitivos (para tipos primitivos, se usan clases envoltorio como Integer, Double, etc.).
  • Permite elementos duplicados: Puede contener valores repetidos.

Funciones Comunes

  • add(elemento): Agrega un elemento al final de la lista.
  • get(indice): Obtiene el elemento en la posición especificada.
  • set(indice, elemento): Reemplaza el elemento en la posición especificada.
  • remove(indice): Elimina el elemento en la posición especificada.
  • size(): Devuelve el número de elementos en la lista.
  • clear(): Elimina todos los elementos de la lista.
  • contains(elemento): Verifica si la lista contiene un elemento específico.
  • isEmpty(): Devuelve true si la lista está vacía.

Ejemplo de Uso


    ArrayList<String> frutas = new ArrayList<>();
    frutas.add("Manzana");
    frutas.add("Banana");
    frutas.add("Naranja");

    txtarea.append("Tamaño de la lista: " + frutas.size()); // 3
    txtarea.append("Primera fruta: " + frutas.get(0)); // Manzana
    frutas.remove(1); // Elimina "Banana"
    txtarea.append("Contiene Manzana? " + frutas.contains("Manzana")); // true

Ejercicio Propuesto

Enunciado:

Desarrolla una aplicación en Java Swing que utilice un ArrayList para gestionar una lista de estudiantes. La interfaz debe incluir lo siguiente:

Campos de entrada:

  • Nombres
  • Edad

Tabla (JTable) para mostrar la lista de estudiantes ingresados.

Botones funcionales:

  • Agregar Estudiante: Al presionar este botón, los datos ingresados se deben agregar a la lista (ArrayList) y reflejarse en la tabla.
  • Eliminar Estudiante: Al seleccionar una fila de la tabla y presionar este botón, el estudiante correspondiente debe eliminarse del ArrayList y la tabla debe actualizarse.

Nota: La tabla debe actualizarse automáticamente después de cada operación.
