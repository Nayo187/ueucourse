¿Qué es Java?

Introducción a Java, su historia y características principales.

Definición

Java es un lenguaje de programación de propósito general, orientado a objetos y diseñado para tener pocas dependencias de implementación.

Historia

Creado por Sun Microsystems en 1995, Java ha evolucionado para convertirse en uno de los lenguajes más populares del mundo.

Características

  • Plataforma independiente
  • Orientado a objetos
  • Robusto y seguro
  • Soporte para concurrencia
