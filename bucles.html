Bucles en Java

Aprende a usar bucles para ejecutar acciones repetitivas.

Tipos de Bucles

  • For: Se usa cuando se conoce el número de iteraciones.
  • While: Se ejecuta mientras una condición sea verdadera.
  • Do-While: Se ejecuta al menos una vez.
